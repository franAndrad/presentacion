# Identity-Core-2026 // Francisco Andrade

Portfolio de ingeniería de alto impacto centrado en el desarrollo Fullstack y sistemas autónomos.

## 🚀 Stack Tecnológico
- **Framework:** Astro 5.0 (Hybrid)
- **UI:** React + Tailwind CSS
- **3D Engine:** React Three Fiber (Three.js)
- **Icons:** Lucide React
- **Typography:** Geist Mono (Engineering Aesthetic)

## 🛠️ Estructura del Proyecto
- `/src/islands`: Componentes interactivos de React (Islands Architecture).
- `/src/layouts`: Estructura base y SEO (JSON-LD).
- `/src/pages`: Rutas y contenido principal.
- `/public/assets`: Certificados y recursos estáticos.

## 💻 Instalación y Desarrollo
```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Compilar para producción
npm run build
```
